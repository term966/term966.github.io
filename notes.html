HTML and CSS Notes

Key terms HTML & CSS:

href = hyperlink reference

P = P tags are to create paragraphs

UL = Unorderd list (list with bullet point)

OL = Ordered list (Number, roman numerals etc..)

H = Heading tags 1-6, most important as 1 and least as 6

img src = Image source - Link to an internal or extarnal image

target_blank = open in a new window

TD = Column in a table

TR = Row in a table

A = a tags wrap arround links

Font-weight = How bold the text is

font-size = Size of the text

font-family = font style - include 2 fonts spearated by , in case the browser can't use the primary font

Text-decoration = underline, strike through etc..

Hover = hover over action psudo class (reliant on an action i.e. mouse hovering over text

span = Span tags wrap arround text to select it

Class = A class allows you to change all text/other as one and ignore all other text instead of just using a tag such as H1

.class = in css you select classes by entering .(class) and then you can style that individual class

style = use style tags to create in internal style sheet within an html doc

Div = division of a page -undefined section tag - not a symantic

Nav = Create a navigation bar

* = Put a star before page setup eg: *{margin: 0; padding: 0; border: 0; outline: 0;}

Background-size: cover = Cover the screen completley

background-repeat: no-repeat = do not repeat background image
